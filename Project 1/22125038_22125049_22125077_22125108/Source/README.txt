# Run test

To run all test will all algorithm, please run:
```
python main.py -f instructions.txt
```

One may freely modify instructions.txt to run multiple batches with different setup.

You can also run task on the terminal with something like this:
```
python main.py --in ./input-01.txt --out ./output-01.txt --sol ./solutions/solution-01.json --strat .*
```

For more details, please run:
```
python main.py -h
```
for helps
